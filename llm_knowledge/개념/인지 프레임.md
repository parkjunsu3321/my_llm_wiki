# 인지 프레임 (Cognitive Frame)

문제를 **의도적으로 다르게 비트는 관점**이다. [[ADHD 스킬]]의 Diverge 단계에서 각 프레임이 독립 Agent 호출의 입력이 된다.

## ADHD에서의 역할

- 한 문제 → N개 프레임으로 분할
- 프레임마다 **별도 추론 프로세스** (맥락 비공유)
- 평가는 Diverge 단계에서 금지, Focus의 critic이 일괄 수행

기본 제공: `adhd-agent` 저장소 `documentation/frames.md`에 **15종**. 사용자 정의 프레임 추가 가능.

## 왜 필요한가

동일 맥락에서 가지를 뻗으면([[Tree-of-Thought]]) 초기 고착이 모든 가지에 번진다. 프레임은 **인지적 렌즈를 바꿔** 탐색 공간을 구조적으로 분리한다.
